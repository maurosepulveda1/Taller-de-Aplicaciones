# Taller-de-Aplicaciones
Esta suite de aplicaciones de Backoffice (con ventanas) ha sido diseñada como una herramienta de ingeniería de software orientada a la Ciencia de Datos Aplicada. Su objetivo principal es permitir a los analistas y científicos de datos generar soluciones propias y personalizadas para el análisis, modelado y proyección de datos, utilizando técnicas avanzadas y modelos estadísticos clásicos.

📋 Requisitos de InstalaciónPara ejecutar cualquiera de los programas de esta suite, asegúrese de tener instalado Python 3.8+ y ejecute el siguiente comando para instalar las dependencias:
pip install customtkinter pandas numpy matplotlib gplearn scikit-learn statsmodels

📥 Cómo descargar los archivos
Existen dos formas principales de obtener este software para ejecutarlo en tu entorno local:

Opción 1: Clonar el repositorio (Recomendado)
Si tienes Git instalado, abre una terminal y ejecuta el siguiente comando:
git clone https://github.com/tu-usuario/nombre-del-repositorio.git
Luego, entra en la carpeta del proyecto:
cd nombre-del-repositorio

Opción 2: Descarga directa (ZIP)
Si no utilizas Git, puedes descargar el código fuente directamente:
Haz clic en el botón verde "Code" situado en la parte superior de esta página.
Selecciona "Download ZIP".
Extrae el contenido del archivo .zip en una carpeta de tu ordenador.
